chat with llms
Running the API:

Linux
    
```export FLASK_APP=chatbot```

```export FLASK_ENV=development```

```flask init-db```

(optional: ```flask fill-db```)

```flask run```

Windows

```set FLASK_APP=chatbot```

```set FLASK_ENV=development```

```flask init-db```

(optional: ```flask fill-db```)

```flask run```


Client:
client can be opened in a browser
